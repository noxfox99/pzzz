# 🛒 Mini Store (Node.js + Express + EJS)

Простой мини-магазин (сахар и соль) с каталогом товаров, просмотром, выбором веса и оплатой (демо).  
Готов к деплою на Vercel.

## 🚀 Функционал
- 📦 Список товаров (`products.json`)
- 🖼 Страница товара
- 💳 Оплата (демо)
- ✅ Страница успеха с адресом

## 🖥 Локально
```bash
npm install
npm start
```

## ☁️ Деплой
1. GitHub → Vercel
2. Deploy 🎉
